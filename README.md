# FinSight AI

> Currently under active development.

FinSight AI is an AI-powered portfolio intelligence platform designed to help investors better understand and manage their portfolios through portfolio analytics, Retrieval-Augmented Generation (RAG), agentic AI workflows, and financial insights.

This project is being developed as a long-term AI engineering portfolio project to strengthen my skills in backend development, Generative AI, system design, and production-ready application development.

https://claude.ai/public/artifacts/10bf3cde-ca80-48c3-ae4a-d4b0792ec7a8 

---

# Project Vision

The long-term goal of FinSight AI is to build an intelligent investment assistant capable of:

- Portfolio management
- Portfolio analytics
- Risk and diversification analysis
- AI-powered financial assistant
- Retrieval-Augmented Generation (RAG)
- Multi-agent AI workflows
- Financial news analysis
- Portfolio import from screenshots
- Natural language portfolio insights

# Purpose

This repository documents my journey of building a production-style AI application while learning modern AI engineering practices.

Rather than focusing on isolated tutorials, the objective of this project is to integrate backend development, AI engineering, Retrieval-Augmented Generation, agentic AI, and system design into a single end-to-end application.

The project is being developed incrementally, with each milestone representing practical implementation and continuous learning.
